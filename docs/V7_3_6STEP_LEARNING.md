## 📚 6 步学习法

### 完整流程

```
1. 学习 → 提取关键信息
    ↓
2. 理解 → 提取核心概念
    ↓
3. 关联 → 与现有知识关联
    ↓
4. 应用 → 思考应用场景
    ↓
5. 反思 → 元认知评估
    ↓
6. 整合 → 保存到知识库
```

---

## 🛠️ learn 工具（6 步版）

### 参数

```typescript
{
  title: string,           // 学习材料标题
  content: string,         // 学习内容
  agent_id?: string        // 智能体 ID（默认 main）
}
```

### 6 步执行流程

**第 1 步：学习**
- LLM 提取关键信息
- 工具层接收 LLM 处理后的内容

**第 2 步：理解**
- LLM 提取核心概念
- 工具层不介入

**第 3 步：关联**
- 工具层通过关键词搜索相关记忆
- 返回关联数量

**第 4 步：应用**
- LLM 思考应用场景
- 工具层不介入

**第 5 步：反思**
- LLM 元认知评估
- 工具层不介入

**第 6 步：整合**
- 工具层保存到 LanceDB
- 包含关联信息

---

### 使用示例

**LLM 调用流程：**
```
用户提供材料
    ↓
LLM 执行 1-2 步（学习/理解）
    ↓
调用 learn 工具
    ↓
工具执行第 3 步（关联）
    ↓
工具执行第 6 步（整合）
    ↓
LLM 执行 4-5 步（应用/反思）
    ↓
返回完整学习结果
```

**调用示例：**
```typescript
learn({
  title: "宏观经济分析",
  content: "LLM 提取后的核心内容..."
})
```

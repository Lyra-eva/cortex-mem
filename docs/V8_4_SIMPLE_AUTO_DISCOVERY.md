## 🛠️ 实施内容

### 核心代码（10 行）

```python
def get_active_agents():
    """从 Redis 扫描所有有记忆的活跃 agent"""
    import redis
    r = redis.Redis(host='localhost', port=6379, decode_responses=True)
    
    # 扫描 episodic:* keys
    agent_keys = r.keys('episodic:*')
    
    # 提取 agent_id
    agents = set(key.split(':')[1] for key in agent_keys)
    
    # 至少包含 main
    agents.add('main')
    
    return sorted(list(agents))
```

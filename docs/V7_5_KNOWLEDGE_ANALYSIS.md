## 🛠️ 新增 analyze_knowledge 工具

### 功能

**基于知识框架分析：**
1. 检索相关知识（最多 10 条）
2. 构建分析框架
3. 保存分析结果
4. LLM 识别模式/缺口

---

### 参数

```typescript
{
  topic: string,           // 分析主题
  framework?: string,      // 知识框架/理论
  compare_with?: string[], // 对比的观点/理论
  agent_id?: string        // 智能体 ID
}
```

---

### 使用示例

**场景 1：基于理论框架分析**
```typescript
analyze_knowledge({
  topic: "经济周期",
  framework: "凯恩斯主义"
})
```

**返回：**
```
✅ 知识分析完成
- 主题：经济周期
- 框架：凯恩斯主义
- 关联记忆：5 条
- 对比观点：0 个
- 分析结果：已保存（LLM 识别模式/缺口）
```

## 🎯 知识分析的核心价值

### 1. 基于框架分析

**作用：**
- 避免碎片化理解
- 建立系统性认知
- 发现理论的适用边界

**示例：**
```
分析"通货膨胀"
    ↓
选择框架：货币主义 vs 凯恩斯主义
    ↓
发现：不同框架有不同解释
    ↓
形成：更全面的理解
```

---

### 2. 对比不同观点

**作用：**
- 避免单一视角
- 发现共识与分歧
- 评估论证质量

**示例：**
```
分析"最低工资"
    ↓
对比：支持 vs 反对
    ↓
发现：就业影响是争议焦点
    ↓
形成：平衡的观点
```

---

### 3. 识别模式

**作用：**
- 发现规律
- 建立预测模型
- 提升迁移能力

**示例：**
```
学习多个经济危机案例
    ↓
识别模式：信贷扩张→泡沫→破裂
    ↓
形成：危机预警指标
```

---

### 4. 发现知识缺口

**作用：**
- 明确学习方向
- 避免盲目学习
- 提升学习效率

**示例：**
```
分析"货币政策"知识
    ↓
发现缺口：不了解传导机制
    ↓
定向学习：货币政策传导机制
```

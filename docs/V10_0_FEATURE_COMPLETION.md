## 📊 实施内容

### P0：智能问答（answer_question）

**功能：** 基于记忆的智能问答

**参数：**
```typescript
{
  question: string,       // 问题内容
  use_memory?: boolean,   // 是否使用记忆（默认 true）
  agent_id?: string       // 智能体 ID
}
```

**使用示例：**
```typescript
answer_question({
  question: "凯恩斯主义的核心观点是什么？",
  use_memory: true
})
```

**流程：**
```
1. 检索相关记忆（semantic 类型，limit 5）
2. 构建上下文
3. 返回答案（LLM 基于上下文生成）
```

---

### P1：复习计划（schedule_review）

**功能：** 基于艾宾浩斯遗忘曲线安排复习

**参数：**
```typescript
{
  memory_id: string,      // 记忆 ID
  intervals?: number[],   // 复习间隔（天），默认 [1,3,7,15,30]
  agent_id?: string       // 智能体 ID
}
```

**使用示例：**
```typescript
schedule_review({
  memory_id: 'mem_123',
  intervals: [1, 3, 7, 15, 30]
})
```

**存储：**
```
Redis: review:{agent_id}:{memory_id}
TTL: 30 天
数据结构：{
  memory_id,
  intervals,
  created_at,
  next_review,
  completed: []
}
```

### P2：学习进度（get_learning_progress）

**功能：** 获取学习进度统计

**参数：**
```typescript
{
  agent_id?: string,      // 智能体 ID
  days?: number           // 统计天数（默认 7）
}
```

**使用示例：**
```typescript
get_learning_progress({
  agent_id: 'main',
  days: 7
})
```

**返回：**
```

记忆总数：1299 条
待复习：1 个
统计周期：7 天
```

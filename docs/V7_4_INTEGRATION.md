## 🛠️ 新增融会贯通工具

### 1. connect_knowledge（知识关联）

**功能：** 关联跨领域知识，构建知识网络

**参数：**
```typescript
{
  topic: string,           // 主题/概念
  agent_id?: string,       // 智能体 ID
  depth?: number          // 关联深度（默认 2）
}
```

**使用示例：**
```typescript
connect_knowledge({
  topic: "经济周期",
  depth: 2
})
```

**返回：**
```
✅ 融会贯通完成
- 主题：经济周期
- 关联记忆：5 条
- 知识网络：已保存
```

**流程：**
```
1. 多跳检索（hop API）
   ↓
2. 构建知识网络
   ↓
3. 保存到 LanceDB
```

---

### 2. apply_knowledge（知识迁移）

**功能：** 思考应用场景，举一反三

**参数：**
```typescript
{
  knowledge: string,       // 已学知识
  scenarios?: string[],    // 应用场景
  agent_id?: string        // 智能体 ID
}
```

**使用示例：**
```typescript
apply_knowledge({
  knowledge: "复利效应",
  scenarios: ["投资理财", "学习成长", "人际关系"]
})
```

**返回：**
```
✅ 知识迁移完成
- 知识：复利效应...
- 应用场景：3 个
- 已保存到程序性记忆
```

**流程：**
```
1. 接收应用场景
   ↓
2. 保存到程序性记忆（procedural）
   ↓
3. 标记为知识迁移
```

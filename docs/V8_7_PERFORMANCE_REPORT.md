## 🎯 优化建议

### P0（本周）- 缓存优化

**目标：** 缓存命中率 11.6% → 50%

**方案：**
```typescript
// before_prompt_build 增加 ctx 缓存检查
const cacheKey = `ctx:${agentId}:${hash(content)}`;
const cached = await redisClient.get(cacheKey);

if (cached) {
  // 缓存命中 → 直接返回（<5ms）
  return { prependSystemContext: cached };
}

// 缓存未命中 → 检索 + 写入缓存
const memories = await searchMemories(content);
await redisClient.setEx(cacheKey, 1800, memories);
```

**预期效果：**
- 缓存命中率：11.6% → 50%+
- 平均延迟：215ms → 50ms

## 📚 学习流程

    ↓

    ↓

    ↓

## 🛠️ learn 工具

### 参数

```typescript
{
  title: string,           // 学习材料标题
  content: string,         // 学习内容
  summary?: string,        // 摘要（LLM 生成）
  keywords?: string[],     // 关键词（LLM 提取）
  url?: string,           // 来源 URL
  tags?: string[],        // 标签
  agent_id?: string       // 智能体 ID（默认 main）
}
```

### 使用示例

**LLM 自动调用：**
```
用户："帮我学习这个宏观经济知识"
    ↓
LLM 提取摘要和关键词
    ↓
learn({
  title: "宏观经济分析",
  content: "GDP 增速、CPI、PPI...",
  summary: "宏观经济指标分析",
  keywords: ["宏观", "经济", "GDP", "CPI"]
})
    ↓
保存到 LanceDB + 关联记忆
```

**返回结果：**
```
✅ 学习完成
- 标题：宏观经济分析
- 关键词：宏观，经济，GDP, CPI
- 关联记忆：3 条
```
